## 🔗 Live Website
https://maruthrein.github.io/

## 🧠 Project Overview

This project is a structured digital environment that combines:

- web development
- narrative design
- introspective content

It is not a simple static website, but a multi-layered experience built through interconnected pages and thematic sections.

## 🧱 Architecture

The project is organized into modular sections:

### 📂 Core Structure
- `index.html` → main entry point
- `components/` → reusable UI elements
- `assets/` → media and resources
- `content/` → thematic pages and sections
- `articoli/` → long-form written content

### 📄 Content Layers

The website explores multiple domains:

- identity (`chi_sono.html`)
- transformation (`dal_sonno_alla_presenza.html`)
- introspection (`il_silenzio_che_guarisce.html`)
- symbolic / conceptual pages (`Ombra.html`, others)

### 🎨 Styling

- Modular CSS structure
- Dedicated styles per section (`grafico.css`, `guru.css`, `luxury.css`, etc.)

### ⚙️ Functionality

- Vanilla JavaScript (`script.js`)
- Structured navigation across pages
- SEO support (`sitemap.xml`)

## 🛠️ Technologies

- HTML5
- CSS3 (modular approach)
- JavaScript (vanilla)

## 🎯 Purpose

To create a hybrid between:
